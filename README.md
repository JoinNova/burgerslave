## 사용법
---
다음 2가지가 필요합니다

* Python 2.X
* BeautifulSoap4

5단계로 쉽게 세트로 업그레이드 해먹을 수 있습니다.

1. git을 clone합니다.
2. clone한 폴더에서 커맨드창을 열고 **python app.py**를 입력합니다
3. 영수증 하단의 설문조사 코드를 그대로 입력합니다 (16자리)
4. 기다리다 보면 확인코드가 나옵니다.
5. 확인코드를 영수증에 적어서 세트업그레이드 해먹으면 끝!
